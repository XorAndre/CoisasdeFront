Conhecendo o HTML

O surgimento do HTML O HTML (abreviação HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação usada para construção de páginas web, a linguagem é o resultado da união entre Hytime e SGML. Diferente de linguagens como PHP, Ruby entre outras que roda somente no lado da camada de back-end o HTML não é uma linguagem lógica, mas antes de falarmos o porquê de não ser uma linguagem de uso lógico vamos entender o que é esse tal de back-end. O back-end é uma camada voltada para trabalho no lado do servidor onde é possível encontrar linguagens como PHP, Java, Ruby, Python, Sql (Banco de dados), Asp.Net e JavaScript (Node-Js que atualmente vem crescendo cada dia mais), mas e daí onde entra o HTML nesta história? As linguagens citadas até então possuem estruturas lógicas if, else e laços de repetições diferente do HTML que não possui! Então quando alguém falar que programa em HTML (Duvide srsr), a maneira correta de dizer é (desenvolver com HTML). Sobre a questão que muitos também dizem programa em HTML5 isso é um modo de dizer que usa JavaScript forçado. Um pouco mais de história Tim Berners-Lee (físico britânico) e criador da linguagem criou o HTML original e outros protocolos baseados no HTTP usando o ambiente de desenvolvimento NeXTSTEP. Nessa época a linguagem era uma coleção de ferramentas usadas para resolver os problemas do físico. A sua invenção combinada com a internet logo ganhou o mundo. As primeiras versões do HTML erão definições de regras sintáticas flexíveis, o que ajudou a quem não tinha familiaridade com a publicação na Web. Com o passar do tempo a rigidez da sintaxe aumentou e por questão de retrocompatibilidade,os navegadores ainda conseguem ler tags que estão longe de um padrão adequado. O passo inicial das definições do HTML foi nos anos 90, inspiradas nas propostas originais de Tim em criar algo parecido com SGML para a internet. O primeiro esboço foi publicado por Berners-Lee e Dan Connolly, e publicada em 1993 como uma aplicação formal para o SGML na IETF. A IETF criou um grupo de trabalho que ficaria no ano seguinte responsável pelo HTML, no ano de 1995 foi lançada a versão a versão 2.0. Desde o ano de 1996 as especificações vêm sido mantidas com auxílio da W3C (World Wide Web Consortium). No ano 2000 a linguagem se tornou uma norma internacional (ISO/IEC 15445:2000). A recomendação da versão 4.01 foi publicada no final do ano 1999 pelo W3C. Uma errata ainda foi lançada em 2001. Em Janeiro de 2008 surgi então a versão mais até então recente o HTML5 que largou de vez as tentativas de semelhança com SGML para então se basear no XML ficando assim XHTML5.
